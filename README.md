# Dementia Subtype Comparison Tool

A research-backed diagnostic comparison tool for identifying patterns in dementia subtypes based on observable behavioral symptoms.

## Features

- Comprehensive symptom checklist organized by category
- Pattern analysis based on NIA-AA, DLB Consortium, and other diagnostic criteria
- Discriminating symptom highlighting for more accurate pattern matching
- Evidence-based research citations for each symptom
- Support for multiple dementia types:
  - Alzheimer's Disease (AD)
  - Vascular Dementia (VaD)
  - Lewy Body Dementia (LBD)
  - Frontotemporal Dementia (FTD)
  - Mixed Dementia

## Development

This project uses Vite + React.

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

## Deployment

This project is configured for deployment on Vercel. The build command and output directory are automatically detected by Vercel.

## Medical Disclaimer

**For discussion with healthcare providers only.** Dementia diagnosis requires comprehensive medical evaluation including cognitive testing, brain imaging, blood work, and clinical assessment. This tool is based on peer-reviewed diagnostic criteria but cannot replace professional evaluation.

## Research Sources

- NIA-AA Alzheimer's criteria (2024)
- DLB Consortium 4th consensus (2017)
- International bvFTD Criteria Consortium
- NINDS-AIREN vascular criteria
